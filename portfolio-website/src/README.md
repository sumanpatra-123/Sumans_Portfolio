# Assets Directory Documentation

This directory contains various assets used in the portfolio website. 

## Contents

- **Images**: Any images used in the portfolio, such as profile pictures, project screenshots, or background images.
- **Icons**: Icons used for navigation or social media links.
- **Fonts**: Custom fonts or font files that enhance the typography of the website.

## Usage

Ensure that all asset files are properly referenced in the HTML and CSS files to maintain the visual integrity of the portfolio.
